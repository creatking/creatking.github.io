实验一 在Github上用Markdown创建技术博客

一、实验目的

  1. 熟悉Github网站
  2. 熟悉Github网站

二、实验任务

  1. 注册Github账号
  2. 创建 Github Pages repository
  3. 发布第一篇博客
  4. 学习Markdown
  5. 利用Hexo生成博客并部署到Github

三、实验步骤与结果

  1. 访问github网站，创建个人账户（creatking）

  2. 开始创建一个新的博客：first_creat

  3. 写入内容步骤

  4. 发布第一篇博客并于同学互相交流

  5. 学习Markdown
    1.访问zybuluo，学习几本书写方法，如下图

    学习使用Typora和Visual Studio Code

  6. 利用Hexo生成博客并部署到github上
    1. 安装 node-v7.7.1-x86.msi
    2. 安装 Git-2.1.2.0-32-bit.exe
    3. 电脑组合键：win+R，并输入 cmd 回车，执行npm install -g hexo-cli
    4. 在cmd命令窗口中，创建Hexo文件夹，执行语句为： hexo init D:\Hexo
    5. 打开Hexo所在文件夹：执行命令：npm install
    6. 在Hexo目录下：执行：hexo g
    7. 在Hexo目录下，执行：hexo s
    8. 重新进入cmd 命令，打开Hexo目录：执行一下三条语句：
      1. npm install hexo-deploter-git –save
      2. git config –global user.emial”522736228@qq.com”
      3. git config –global user.name”creatking”
    9. 最后一步执行下列语句
      1. hexo d -g
      2. 在跳出的对话框中，输入github的用户名和密码。
    10. 部署成功：访问creatking.github.io

四 、实验小结

1、实验的操作性很强，要我们自己去操作实现，是很好的锻炼。
